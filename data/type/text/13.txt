
spellcheck
Compare spellchecking algorithms for texts in a dataset
Install

git clone https://github.com/smilli/spellcheck.git
(recommended) virtualenv venv && source venv/bin/activate
pip install -r requirements.txt
Install the Enchant library.  If on Mac OS X, you can just do brew install enchant.

Usage
To run the spellchecker statistics use python spellchecker_stats.py -d <path to dataset file>
