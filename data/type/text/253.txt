
HTR-using-Pytorch
Implementation of Handwritten Text Recognition on IAM dataset using Pytorch.
The architecture is based on https://github.com/githubharald/SimpleHTR.
Achieved probability of 2.02E-22 on validation set and 3.45E-05 on train set after 123 epochs.
Use test-model file in order to see output vs labels.
